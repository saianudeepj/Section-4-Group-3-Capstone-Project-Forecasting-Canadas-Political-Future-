# Section-4-Group-3-Capstone-Project
## Forecasting Canada's Political Future
“Prediction is not just about seeing the future—it’s about empowering decisions in the present.” — Anonymous

In the digital age, the landscape of political forecasting has shifted dramatically. No longer confined to opinion polls and expert speculation, we now have the tools to extract insights from massive volumes of structured and unstructured data. This evolution is powered by Machine Learning (ML) and Data Analysis, two fields that have revolutionized how we interpret information and anticipate outcomes.
Machine Learning refers to algorithms that learn patterns from historical data and apply that learning to make predictions or decisions. It thrives in scenarios where complexity and data volume surpass traditional analytical methods. Data Analysis, on the other hand, forms the foundation by transforming raw data into actionable knowledge through processes like cleaning, exploration, visualization, and statistical inference.
In this capstone project—Forecasting Canada's Political Future—we combine these two disciplines to address a multifaceted problem: understanding how past voting behaviours and modern digital discourse shape electoral outcomes.
The project is divided into two distinct yet interconnected parts:
•	Part 1: Structured Data Analysis
We begin by examining historical Canadian federal election data sourced from Harvard Dataverse. After preparing and exploring the data, we implement multiple ML models to predict vote percentages and candidate outcomes. Visualization tools like Tableau are used to enhance interpretability and highlight key patterns. Among the models tested, XGBoost emerged as the most accurate and reliable for this classification task.
•	Part 2: Sentiment Analysis of Election Discourse
In the second phase, we analyze tweets and news articles collected via GitHub repositories from the 2019 Canadian federal election. Using Natural Language Processing (NLP) techniques—especially the VADER lexicon—we assign sentiment scores to public commentary. These insights are used to identify political affiliations and gauge public opinion. Here again, XGBoost proves to be the best-performing model in classifying sentiments across textual data.
Together, these parts create a holistic framework for forecasting electoral trends in Canada, blending historical insight with real-time public sentiment. The outcome is not only a powerful predictive tool but also a demonstration of how ML and data science can be effectively applied in political research and strategy.
